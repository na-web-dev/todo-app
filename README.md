# Todo App

React + TypeScriptで作成したシンプルなTodoアプリ
※React確認用で作成したためCSSはなし

## 機能

- タスク追加
- タスク完了チェック（チェックボックスON/OFF）
- 完了タスクの削除
- 未完了タスク数の表示

## 技術スタック

- React
- TypeScript
- Vite

## 実行方法

以下を実行
```bash
npm install
npm run dev
```

## memo

- useState を使った状態管理
- map/filter を使ったデータ操作
- TypeScriptで型定義を意識
