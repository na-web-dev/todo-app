# Todo App

React + TypeScriptで作成したシンプルなTodoアプリ
※React確認用で作成したためCSSはなし

## 機能

- タスク追加
- 初期表示(タスク0件)メッセージ表示
- タスク完了チェックON/OFF
- 追加したタスクの全件数表示
- (チェックありの場合のみ)チェック中タスクの件数表示
- チェック中タスクの削除
- 全選択 / 全解除機能の実装
- 選択状態に応じたボタン制御(disabled)
- 入力後のフォーカス制御

## 技術スタック

- React
- TypeScript
- Vite

## 実行方法

以下を実行

```bash
npm install
npm run dev
```

## memo

- useState を使った状態管理
- map/filter を使ったデータ操作
- TypeScriptで型定義を意識
